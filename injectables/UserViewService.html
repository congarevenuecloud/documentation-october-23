Description

This service is a work in progress User view service returns the configurable view of cart item fields to be displayed for a user on cart page.

Usage

import { UserViewService, AObjectService} from '@congarevenuecloud/ecommerce';

constructor(private userViewService: UserViewService) {}

// or

export class MyService extends AObjectService {
private userViewService: UserViewService = this.injector.get(UserViewService);
}

getUserView
getUserView(viewId: string)

This method fetches the list of display columns for cart item fields, matching the default user view.

Example:

import { UserViewService } from '@congarevenuecloud/ecommerce';

constructor(private userViewService: UserViewService) {
this.subscriptions.push(this.userViewService.getUserView().subscribe(res => {....}));
}
Parameters :
Name Type Optional Default value Description
viewId string No null

string identifier, an optional parameter representing the user view.
Returns : Observable<Array<UserView>>

an observable containing list of display columns for cart item fields.
